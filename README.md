# 🚀 SmartTaskManager

A modern Desktop Task Management Application built with **Java** and **JavaFX**.

## ✨ Key Features
* **Modern GUI:** Designed using JavaFX for a clean user experience.
* **Database Integration:** Utilizes SQLite for saving tasks.
* **Concurrency:** Implements multi-threading for smooth performance.

## 🛠️ Technologies Used
* **Language:** Java 
* **Framework:** JavaFX
* **Build Tool:** Maven
* **Database:** SQLite
